cmatrix-win

Matrix digital rain for the Windows terminal, written in C. Lightweight, fast, no dependencies.

Windows 10+ C ANSI / VT100 UTF-8 katakana open source
⇓ Download .exe View on GitHub

Usage

cmatrix [-b] [-B] [-c] [-r] [-s speed]
OptionDescription
-bBold characters
-BAll bold (overrides -b)
-cKatakana characters (needs UTF-8 font)
-rRainbow colors
-s 1–9Speed — 1 = slow, 9 = fast (default 5)

Press any key to exit.

Requirements

Windows 10 or later (requires ANSI/VT terminal support).

A terminal with a monospace font — Windows Terminal is recommended.

For -c (katakana): a font that includes half-width katakana, such as MS Gothic or Cascadia Code.

Running the downloaded .exe

Windows may show a "Windows protected your PC" SmartScreen warning on first launch. This happens because the executable is not commercially code-signed.

To run it anyway: click More info, then Run anyway.

The app is fully open source — you can review the source and build it yourself if you prefer.

Building from source

Requires MinGW (gcc) or MSVC.

build.bat

Or manually with MinGW:

gcc -O2 -static -o cmatrix.exe cmatrix.c
Note: Make sure C:\MinGW\bin is in your PATH before running gcc.
